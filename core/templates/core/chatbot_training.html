Help & Guidelines
Pro Tips for Training SWEKEER
  • Be Specific: The more specific the question, the better SWEKEER can match similar user queries.
  • Add Variations: Create multiple Q&A pairs for different ways users might ask the same question.
  • Use Keywords: Keywords help boost matching accuracy for related questions.
  • Organize with Intent: Group related Q&As by intent (greeting, event_info, pricing, etc.).
  • Keep Active: Deactivate outdated pairs instead of deleting them for future reference.
